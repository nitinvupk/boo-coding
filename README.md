# Setup Details 
## System Requirements

- NodeJs
- Express
- MongoDB

## setup
```
extract your zip file
> cd boo-coding-test-task/server
> npm install
> npm start
```

## test cases run command
```
npx jest
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
